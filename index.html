Free version of Wandi-SSAL (Secure Software Adaptable Logger) is a C-Program application library that validates and protects application log messages during development and deployment

Software applications generate log messages to help monitor, analyze and diagnose execution flow. Often these messages contain critical and sensitive information about the behavior and use of data. Current approaches do not do enough to protect this information. Wandi-SSAL provides a rich set of capabilities that can be dynamically configured to address this and many other logging limitations and challenges. Wandi-SSAL capabilities is based on a Patent Software Design Technology that:

  1. uses a simple C-Program print() style function interface.
  2. analyzes for erroneous and malicious use.
  3. protects before and after saved to disk.
  4. ensures confidentiality, integrity, and authenticity.
